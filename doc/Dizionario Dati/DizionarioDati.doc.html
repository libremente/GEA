  1. Introduzione

Il seguente documento fornisce una visione dettagliata del modello dati del repository Alfresco.

Nei successivi capitoli viene fornito un elenco delle entità che compongono il modello dati ed una visione più dettagliata che ne evidenzia le relazioni gerarchiche e le loro proprietà (metadati).

  1. Dettaglio Entità e proprietà.

Di seguito lo schema dettagliato delle entità con i loro attributi. Data la complessità del modello e delle entità che lo compongono si è adottato un tipo di disegno che ne descrive le relazioni gerarchiche.

Gli attributi usati per tutte le entità sono i seguenti :

  • Nome : Il nome univoco di ogni entità.
  • Titolo : Il titolo descrittivo delle entità.
  • Parent : Entità padre e quindi contenitore.
  • Attributi : Serie di proprietà.
  • Proprietà : L’insieme delle proprietà di una entità.
  • Tipo : Tipologia di metadato.
  • Associazioni : Associazioni tra entità.
  • Source : Definisce la cardinalità per le associazioni.
  • Target : Entità associata.
  • Mandatory- aspects : Serie di metadati trasversali a tutte le entità.
